3. Texture Mapping

I use image textures for the first sphere and the second cylinder. Checker patterns can be created by arbitrary number of squares per row and per column. The texture for the third cylinder is generated by perlin noise.

4. Bump Mapping

As shown above, primitives of the last row use normal maps as follows, as well as the wall behind:

The third and forth spheres use the same materials, but the forth one have uneven surface due to normal mapping.
To simulate the uneven texture of brick walls, I use Perlin Noise to turbulence the surface normal of objects.
